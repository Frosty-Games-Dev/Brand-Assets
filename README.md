# Brand Assets

Dépôt officiel des éléments de marque.

Ce repository contient les logos, visuels, textes et ressources de communication
utilisés pour la marque.

## Contenu
- `/logos` : logos officiels (SVG, PNG, variations)
- `/visuals` : visuels, illustrations, bannières
- `/texts` : textes officiels, descriptions, slogans
- `/guidelines` : charte graphique et règles d’utilisation

## Usage
Ces éléments sont destinés à :
- la communication officielle
- les supports marketing
- les partenaires autorisés

Toute utilisation doit respecter la charte graphique fournie.
